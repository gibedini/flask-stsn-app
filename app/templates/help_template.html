Message template help

Templates are regular text files with placeholders for database fields. Enclose the desired database field in double curly parentheses {{ '{{' }} like this {{ '}}' }} within the message text: it will then be replaced by the actual field value in the rendered message.
Currently the following fields are recognized by the template renderer:

FieldDescription
titoloname prefix, e.g. "Gent.mo Sig." or "Gent.ma Sig.ra"
nomefirst name
cognomelast name
ultimo_annoyear of last subscription fee payment
anno da saldarefirst year for which payment is due
a_debitoamount due, in Euros

So, if you enter {{ '{{' }} titolo {{ '}}' }} in the template, it will be rendered as "Gent.mo Sig." or "Gent.ma Sig.ra" based on member sex.
To control message formatting, a template can include regular HTML formatting tag. To create a new template, press the "New template" button. {% endblock %}
